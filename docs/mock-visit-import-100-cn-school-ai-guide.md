# AI 演示病例清单

导入文件：`docs/mock-visit-import-100-cn-school.json`

这份 100 条数据里，下面几条专门做了 AI 演示强化，适合在“就诊详情 / AI 辅助”页面直接打开演示。

## 1. 高风险分诊演示

- `20262013`
  - 症状含 `胸痛`、`呼吸困难`
  - 适合演示：`症状结构化`、`智能分诊`
  - 预期效果：高风险，建议转诊医院

- `20262018`
  - 症状含 `晕厥`
  - 适合演示：`症状结构化`、`智能分诊`
  - 预期效果：高风险或紧急处理建议

- `20262048`
  - 症状含 `抽搐`
  - 适合演示：`症状结构化`、`智能分诊`
  - 预期效果：高风险，建议立即转诊

- `20262073`
  - 症状含 `胸痛`、`呼吸困难`
  - 适合演示：`一键生成建议`
  - 预期效果：直接出现高优先级分诊结论

- `20262093`
  - 运动后 `胸痛`、`呼吸困难`
  - 适合演示：`智能分诊`
  - 预期效果：建议医院评估

## 2. 中风险分析演示

- `20262001`
  - 三个常见症状：发热、咽痛、乏力
  - 适合演示：普通感冒类 `症状结构化` 和 `药品推荐`

- `20262008`
  - 三个症状：发热、肌肉酸痛、咳嗽
  - 适合演示：中风险 `智能分诊`

- `20262089`
  - 三个症状：皮疹、瘙痒、低热
  - 适合演示：过敏场景 `症状结构化`

## 3. 药物相互作用演示

- `20262020`
  - 处方：`cetirizine + chlorpheniramine`
  - 适合演示：`相互作用检查`
  - 预期效果：提示嗜睡风险增加

- `20262030`
  - 处方：`aspirin + ibuprofen`
  - 适合演示：`相互作用检查`
  - 预期效果：提示胃肠道不良反应风险增加

- `20262089`
  - 处方：`acetaminophen + ibuprofen`
  - 适合演示：`相互作用检查`
  - 预期效果：提示低等级重复止痛风险

## 4. 推荐演示顺序

1. 先打开 `20262001`，演示普通病例的 `症状结构化 -> 智能分诊 -> 药品推荐`
2. 再打开 `20262013`，演示高风险病例的 `智能分诊`
3. 再打开 `20262020` 或 `20262030`，演示 `药物相互作用检查`
4. 最后打开 `20262048`，演示“一键生成建议”在高风险病例下的效果
