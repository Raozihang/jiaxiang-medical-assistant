# 06｜数据模型（PostgreSQL）

## 1. 字段通用约定
- 主键：`uuid`（推荐 UUID v7）；
- 审计字段：`created_at`、`updated_at`；
- 软删除（可选）：`deleted_at`；
- 时间类型：`timestamptz`。

## 2. 核心表

### 2.1 students（学生）
| 字段 | 类型 | 说明 |
|---|---|---|
| id | uuid | 主键 |
| student_id | varchar(32) | 学号，唯一 |
| name | varchar(64) | 姓名 |
| class_id | uuid | 班级 ID |
| grade | varchar(16) | 年级 |
| created_at | timestamptz | 创建时间 |
| updated_at | timestamptz | 更新时间 |

### 2.2 visits（就诊记录）
| 字段 | 类型 | 说明 |
|---|---|---|
| id | uuid | 主键 |
| student_id | uuid | 学生 ID |
| doctor_id | uuid | 校医 ID |
| symptoms | jsonb | 症状描述 |
| diagnosis | text | 诊断结果 |
| prescription | jsonb | 处方信息 |
| destination | varchar(32) | 去向 |
| created_at | timestamptz | 就诊时间 |
| updated_at | timestamptz | 更新时间 |

### 2.3 medicines（药品）
| 字段 | 类型 | 说明 |
|---|---|---|
| id | uuid | 主键 |
| name | varchar(128) | 药品名称 |
| specification | varchar(128) | 规格 |
| stock | integer | 库存 |
| expiry_date | date | 有效期 |
| warnings | jsonb | 禁忌/不良反应 |
| created_at | timestamptz | 创建时间 |
| updated_at | timestamptz | 更新时间 |

## 3. 索引建议
- `students(student_id)` 唯一索引；
- `visits(student_id, created_at desc)` 复合索引；
- `visits(created_at)` 普通索引（报表统计）；
- `medicines(name)` 普通索引；
- `medicines(expiry_date)` 普通索引（效期预警）。

## 4. 数据约束建议
- 库存不得为负数（`stock >= 0`）；
- 就诊记录必须绑定学生与医生；
- 关键枚举字段（如 `destination`）限制可选值。
