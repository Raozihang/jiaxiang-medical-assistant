# 嘉祥智能医务室助手｜项目概览

## 项目简介
嘉祥智能医务室助手是一个面向校园医务室的数字化管理系统，目标是提升接诊效率、优化医务流程，并支持健康数据沉淀与分析。

## 核心目标
- 标准化学生就诊登记与电子病历流程
- 支持医务人员高效完成诊疗与药品管理
- 打通学生、家长、教师、管理者的信息联动
- 为校内健康决策提供数据支撑
