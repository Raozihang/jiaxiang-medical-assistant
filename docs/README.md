# 嘉祥智能医务室助手｜开发文档总览

## 项目定位
嘉祥智能医务室助手是面向中小学医务室的数字化系统，围绕「就诊登记 → 病历管理 → 药品库存 → 安全追踪 → 数据分析」建设一体化平台。

## 快速开始
### 前端（React + TypeScript）
```bash
cd frontend
npm install
npm run dev
```

### 后端（Go + Gin）
```bash
cd backend
go mod tidy
go run ./cmd/server
```

- 需求变更必须先更新 PRD，再同步更新对应技术文档。
