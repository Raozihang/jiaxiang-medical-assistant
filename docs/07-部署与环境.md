# 07｜部署与环境

## 1. 本地开发环境
- Node.js 20+
- npm 10+
- Go 1.22+
- PostgreSQL 14+
- Redis 7+（P0 可选）

## 2. 环境变量

### 2.1 前端（`frontend/.env.example`）
```env
VITE_API_BASE_URL=http://localhost:8080/api/v1
VITE_APP_TITLE=嘉祥智能医务室助手
```

### 2.2 后端（`backend/.env.example`）
```env
APP_ENV=development
APP_PORT=8080
APP_NAME=medical-assistant
DB_HOST=127.0.0.1
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=postgres
DB_NAME=medical_assistant
DB_SSLMODE=disable
```

## 3. 本地启动流程
1. 启动 PostgreSQL；
2. 启动后端服务；
3. 启动前端开发服务；
4. 打开前端页面验证联通性。

## 4. 生产部署建议
- Nginx 反向代理 + 后端容器；
- PostgreSQL 使用独立实例并开启备份；
- 启用 HTTPS 与访问审计日志；
- 建议最小双环境：`staging`、`production`。

## 5. 运维基线
- 结构化日志（JSON）
- 健康检查接口（`/api/v1/healthz`）
- 关键指标：请求量、错误率、接口耗时、数据库连接数
